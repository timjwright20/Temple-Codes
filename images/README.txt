PHOTOS FOLDER
=============

To replace the generic placeholder with a real temple photo, drop a JPG
file into this folder named with the temple's 5-letter abbreviation in
lowercase, for example:

    slake.jpg    (Salt Lake Temple)
    provo.jpg    (Provo Utah Temple)
    nzeal.jpg    (Hamilton New Zealand Temple)
    lange.jpg    (Los Angeles California Temple)
    spaul.jpg    (São Paulo Brazil Temple)

The app automatically falls back to _placeholder.svg whenever a photo
for that abbreviation isn't found, so you can add photos one at a time
whenever you're ready.

Recommended: 1200 x 800 JPG, landscape orientation.

Full list of abbreviations and temple names lives in data/temples.json.
